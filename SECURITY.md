# Security Policy

## Reporting a Vulnerability

Please do **not** report security vulnerabilities through public GitHub issues.

To report a vulnerability, please contact:
- **Email:** (update with your contact email)
- **Response time:** Within 48 hours for acknowledgment

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact assessment
- Any suggested fixes (optional)

## Supported Versions

| Version | Supported |
|---------|-----------|
| Latest  | Yes       |

## Security Measures

This project uses:
- SSH-signed commits (all commits are cryptographically verified)
- gitleaks pre-commit scanning (no secrets committed)
- Dependabot alerts (where applicable)
